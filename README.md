# Blinkit Regional Stores Analysis
## Project Objective 
The Regional Sales manager of Blinkit Commerce Private Limited wants to now the performance of warehouse and outlet, for groceries in the initial year of establishment across various outlets established from 2011 to 2022.

## Dataset Used
- <a href="https://github.com/i4zeeshan/Blinkit_Regional_Store_Analysis/blob/master/dataset.xlsx">Dataset</a>

## Questions (KPIs)
- Total Revenue Generated
- Total items traded
- Customer Satisfaction
- Consumer's Preference by Fat contents in the order
- Performance of channel across outlet location and outlet size

- Dashboard Overview <a href="https://github.com/i4zeeshan/Blinkit_Regional_Store_Analysis/blob/master/Dashboard.pbix">View Dashboard</a>

## Process
- Replace the data that were written differently to make it consistent in MS Excel
- Certain value in a column was null, So the data in the column was not considered for analsis.
- Create the Dashboard with KPIs according to the Question asked
